# Mint-Frontend

Welcome to Mint-Frontend repository! The purpose of the service is to help your finance. This repository is in charge of the frontend. If you want to see the server, Let's move to the [mint-server repository](https://github.com/junngo/mint-server).

## Install & Run

```
git clone https://github.com/junngo/mint-frontend.git
yarn install
yarn start
```
